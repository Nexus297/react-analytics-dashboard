# Class Performance Analytical Dashboard

A React-based analytical dashboard designed to evaluate class performance using static result data. This web app provides tailored data visualizations for different academic roles to help analyze student and subject performance.

## Live Demo:
[https://nexus297.github.io/react-analytics-dashboard/](https://nexus297.github.io/react-analytics-dashboard/)

## Features:
* **Faculty View:** Analyzes the highest achieving students in the current batch using a Bar Chart.
* **HOD View:** Compares class averages and highest scores across different subjects to identify strong/weak areas.
* **Student View:** Displays the overall grade distribution (S, A, B, C, D, E, F) of the class using a Pie Chart.
* **Theme:** Styled with the 2025 "Eco-Analytical" color palette for a modern, sustainable aesthetic.

## Built With:
* [React](https://reactjs.org/) (Create-React-App)
* [Recharts](https://recharts.org/) (Data Visualization)
* Vanilla CSS
